# CMH_Tests

This folder contains scripts used to perform Cochran-Mantel-Haenszel (CMH) tests comparing allele frequency changes between Cycle 1 and Cycle 15 for Control, Moderate Ethanol, and High Ethanol populations. 

## Folder contents

- **Scripts**: All R scripts required to perform CMH tests.
- **Inputs**: The primary SNP table used for these analyses is available in the corresponding Dryad folder associated with the project).
- **Outputs**: Results generated by these scripts (e.g., CMH test results) are available in the matching Dryad folder.
